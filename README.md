In oder to create headings you can use "#" sign for it. 

one "#" translates to h1

# This is h1 tag.

two "##" translates to H2

## This is h2 tag.

three "###" translates to H3
### This is h3 tag. 

four "####" translates to h4

#### This is h4 tag.
